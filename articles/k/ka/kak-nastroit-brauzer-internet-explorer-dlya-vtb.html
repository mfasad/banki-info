Как настроить Internet Explorer для работы с ВТБ-Онлайн в 2026 году

Несмотря на то, что Microsoft Internet Explorer официально прекратил поддержку в 2022 году, многие клиенты ВТБ до сих пор сталкиваются с необходимостью его настройки. Причина проста: некоторые корпоративные системы банка, внутренние порталы для сотрудников или устаревшие версии ВТБ-Онлайн могут требовать именно этого браузера для корректной работы. Более того, в отдельных регионах России до сих пор используются терминалы самообслуживания с IE в качестве основного браузера.

В этой статье мы разберём актуальные на 2026 год настройки Internet Explorer 11, которые позволят избежать ошибок при авторизации в личном кабинете ВТБ, работе с электронными документами или использовании банковских сервисов через устаревшие интерфейсы. Особое внимание уделим параметрам безопасности, без которых банк может блокировать доступ, а также распространённым проблемам вроде "белого экрана" или ошибки DLG_FLAGS_SEC_CERT_DATE_INVALID.

Важно: если вы используете обычный компьютер (не корпоративный), мы рекомендуем перейти на Microsoft Edge в режиме IE Mode — это официальная замена от Microsoft, поддерживаемая ВТБ. Инструкция по настройке этого режима также приведена ниже.

Почему ВТБ может требовать Internet Explorer в 2026 году

На первый взгляд требование использовать IE кажется абсурдным: браузер давно не обновляется, имеет уязвимости и не поддерживает современные веб-стандарты. Однако у этого есть объективные причины:

1. Унаследованные системы. Некоторые внутренние сервисы ВТБ (например, для юридических лиц или корпоративных клиентов) разрабатывались 10+ лет назад и оптимизированы именно под ActiveX и VBScript — технологии, которые работают только в IE. Перенос таких систем на современные платформы требует миллионных вложений и времени.

2. Банковские терминалы. В отделениях ВТБ до сих пор стоят терминалы самообслуживания на базе Windows 7 Embedded с предустановленным IE 11. Эти устройства не обновляются из-за сертификации ЦБ РФ — любое изменение ПО требует повторной проверки регулятором.

3. Электронная подпись и криптография. Старые версии плагинов для работы с ЭЦП (например, КриптоПро CSP) могут корректно функционировать только в Internet Explorer. Новые браузеры блокируют устаревшие расширения по соображениям безопасности.

⚠️ Внимание: Если вы не являетесь корпоративным клиентом ВТБ или не работаете с терминалами в отделении, использовать Internet Explorer для повседневного доступа к ВТБ-Онлайн небезопасно. Банк официально рекомендует современные браузеры (Chrome, Edge, Firefox) для частных клиентов.

Прежде чем приступать к настройке, проверьте, действительно ли вам нужен IE. Попробуйте войти в личный кабинет через Microsoft Edge в режиме совместимости:

  • 🔹 Откройте Edge, нажмите … → Другие инструменты → Режим Internet Explorer (IE Mode).
  • 🔹 Введите адрес online.vtb.ru и проверьте, загружается ли страница.
  • 🔹 Если появится ошибка, вернитесь к этой инструкции.

Подготовка браузера: обновление и базовые настройки

Даже если Internet Explorer 11 установлен на вашем ПК, его необходимо привести в актуальное состояние. Устаревшие компоненты могут вызывать ошибки при работе с ВТБ-Онлайн, особенно если речь идёт о защищённом соединении (HTTPS) или плагинах для ЭЦП.

Следуйте этому алгоритму:

  1. Проверьте версию браузера: нажмите ⚙ → Сведения о Internet Explorer. Должна быть указана версия 11.576.18362.0 или выше. Если версия ниже — обновите Windows.
  2. Установите последние обновления безопасности: 
    Пуск → Параметры → Обновление и безопасность → Проверка наличия обновлений

    Обратите внимание на пакеты с пометкой KBxxxxxxx (например, KB5003690 для IE).

  3. Сбросьте настройки браузера до заводских: 
    ⚙ → Свойства обозревателя → Дополнительно → Сброс

    Это удалит все расширения и временные файлы, которые могут конфликтовать с сайтом ВТБ.

После сброса перезагрузите компьютер. Это критично: некоторые компоненты IE (например, MSHTML) обновляются только после перезагрузки.

Настройка параметров безопасности для ВТБ-Онлайн

Основная проблема при входе в личный кабинет ВТБ через IE — блокировка по причине "небезопасного соединения". Это происходит из-за того, что банк использует современные протоколы TLS 1.2/1.3, а в браузере по умолчанию могут быть включены устаревшие SSL 3.0 или TLS 1.0, которые ВТБ отключил в целях безопасности.

Чтобы исправить это, выполните следующие шаги:

  1. Откройте ⚙ → Свойства обозревателя → Дополнительно.
  2. В разделе Безопасность найдите блок Использовать TLS 1.0/1.1/1.2 и установите галочки так:
    • ☑ Использовать TLS 1.2 (обязательно)
    • ☑ Использовать TLS 1.1 (опционально, но рекомендуется)
    • ☐ Использовать TLS 1.0 (снимите галочку — этот протокол небезопасен)
    • ☐ Использовать SSL 3.0 (обязательно снимите галочку)
  • Нажмите Применить и перезапустите браузер.

    • Если после этих действий сайт ВТБ всё равно не открывается, проверьте зону безопасности:

    1. Перейдите в Свойства обозревателя → Безопасность.
    2. Выберите зону Надёжные сайты и нажмите Сайты.
    3. Добавьте в список: 
      https://online.vtb.ru

      https://*.vtb.ru

      Убедитесь, что флажок Для всех сайтов этой зоны требуется проверка серверов (https:) снят.

    netsh winhttp reset proxy

    certmgr.msc

    В открывшемся окне удалите все сертификаты с истекшим сроком действия.-->

    Настройка совместимости для старых версий ВТБ-Онлайн

    Если вы работаете с корпоративным порталом ВТБ или устаревшей версией личного кабинета (например, для юридических лиц), может потребоваться включить режим совместимости. Это эмулирует поведение более ранних версий IE (например, IE 8 или IE 9), что иногда необходимо для корректной работы скриптов.

    Как включить режим совместимости:

    1. Откройте сайт ВТБ в Internet Explorer.
    2. Нажмите на иконку 🔧 (Инструменты) → Режим предприятия или Режим совместимости.
    3. Если пункта нет в меню, добавьте сайт в список совместимости вручную: 
      ⚙ → Свойства обозревателя → Совместимость → Настройки

      Добавьте online.vtb.ru и установите галочку Отображать все сайты в режиме совместимости.

    Для корпоративных пользователей ВТБ может потребоваться дополнительная настройка документа X-UA-Compatible. Если при открытии внутреннего портала вы видите ошибку вроде "Этот сайт оптимизирован для IE8", выполните следующее:

    • 📝 Откройте Редактор реестра (Win + R → regedit).
    • 🔍 Перейдите по пути: 
      HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_BROWSER_EMULATION
    • 🖱️ Создайте параметр DWORD (32 бита) с именем iexplore.exe и значением 8000 (это соответствует режиму IE 8).
    ⚠️ Внимание: Редактирование реестра может привести к нестабильной работе системы. Перед изменениями создайте точку восстановления. Если вы не уверены в своих действиях, обратитесь в службу поддержки ВТБ по телефону 8 800 100-24-24.

    Установка и настройка плагинов для электронной подписи

    Одной из главных причин использования Internet Explorer в ВТБ является работа с электронной цифровой подписью (ЭЦП). Банк поддерживает несколько типов подписей, но наиболее распространённая — КриптоПро CSP. Для её корректной работы в IE необходимо установить и настроить следующие компоненты:

    Компонент Версия Ссылка для скачивания Примечание
    КриптоПро CSP 5.0 или 4.0 cryptopro.ru Требуется лицензия (бесплатная для физлиц)
    Плагин КриптоПро ЭЦП Browser plug-in 2.0.12673 cryptopro.ru/products/cades/plugin Поддерживает только IE и Edge в IE Mode
    Корневые сертификаты УЦ ВТБ Актуальные vtb.ru (раздел "Электронные сервисы") Устанавливаются в "Личное" хранилище
    Java (при использовании старого портала) 8 Update 331 java.com Только для юридических лиц

    После установки плагинов выполните их настройку:

    1. Откройте Панель управления → КриптоПро CSP.
    2. Перейдите на вкладку Сервис и проверьте, что ваш сертификат ЭЦП отображается в списке.
    3. В Internet Explorer разрешите запуск активного содержимого: 
      Свойства обозревателя → Безопасность → Надёжные сайты → Уровень по умолчанию → Включить ActiveX

    Если после установки плагинов при попытке подписать документ появляется ошибка 0x80070005 (Отказано в доступе), проверьте права на папку C:\Program Files (x86)\Crypto Pro — у вашей учётной записи должны быть права на запись.

    Что делать, если КриптоПро не видит сертификат?

    1. Проверьте, что сертификат установлен в хранилище "Личное" (через certmgr.msc).

    2. Убедитесь, что в IE включён параметр Проверка отзыва сертификатов издателя (вкладка "Дополнительно").

    3. Переустановите плагин CryptoPro CAdES Browser Plug-in с правами администратора.

    4. Если используете Windows 10/11, попробуйте запустить IE от имени администратора (правый клик по ярлыку → "Запуск от имени администратора").

    Решение распространённых ошибок в Internet Explorer

    Даже после правильной настройки IE могут возникать ошибки. Ниже приведён список самых частых проблем при работе с ВТБ-Онлайн и способы их решения.

    1. Ошибка "Сертификат безопасности сайта не является доверенным" (DLG_FLAGS_SEC_CERT_DATE_INVALID)

    Причина: на вашем ПК устарели корневые сертификаты или неправильно настроено время.

    Решение:

    • 🕒 Проверьте дату и время на компьютере (должны совпадать с текущими).
    • 🔄 Обновите корневые сертификаты Windows: 
      Пуск → Параметры → Обновление и безопасность → Дополнительные параметры → Обновления для других продуктов Microsoft
    • 🔗 Вручную установите сертификат УЦ ВТБ (скачать можно на официальном сайте).

    2. Белый экран или пустая страница после входа

    Причина: блокировка скриптов или устаревший JavaScript.

    Решение:

    • 🛠️ Включите выполнение скриптов: 
      Свойства обозревателя → Безопасность → Надёжные сайты → Уровень по умолчанию → Разрешить активное содержимое
    • 🔄 Очистите кэш IE: 
      ⚙ → Безопасность → Удалить журнал обозревателя → Удалить всё (включая "Временные файлы Интернета")
    • 🔄 Отключите AdBlock или другие расширения, которые могут блокировать контент.

    3. Ошибка "Невозможно отобразить страницу" (INET_E_RESOURCE_NOT_FOUND)

    Причина: проблемы с DNS или прокси-сервером.

    Решение:

    • 🌐 Проверьте подключение к интернету.
    • 🔧 Сбросьте настройки сети: 
      Командная строка (администратор) → netsh winsock reset
    • 🔄 Отключите прокси (если не используете): 
      Свойства обозревателя → Подключения → Настройка сети → снять галочку с "Использовать прокси-сервер"
    💡

    Если ни один из способов не помог, попробуйте использовать Microsoft Edge в режиме IE Mode — это официально поддерживаемый ВТБ способ работы со старыми системами без рисков безопасности.

    Альтернативы Internet Explorer для работы с ВТБ

    Если настройка Internet Explorer кажется слишком сложной или вы опасаетесь за безопасность, рассмотрите альтернативные способы доступа к сервисам ВТБ:

    • 🌐 Microsoft Edge в режиме IE Mode:

      Это встроенный режим совместимости, который эмулирует IE 11 внутри современного браузера. Как включить: 

      Edge → … → Другие инструменты → Режим Internet Explorer (IE Mode) → Перезапустить

      После этого все сайты из списка enterpriseModeSiteList будут открываться в режиме IE.

    • 📱 Мобильное приложение ВТБ-Онлайн:

      Доступно для iOS и Android. Поддерживает все функции личного кабинета, включая платежи и переводы. Скачать можно по ссылке: vtb.ru/personal/mobile/.

    • 💻 Веб-версия в современных браузерах:

      Для большинства клиентов (физических лиц) сайт online.vtb.ru корректно работает в Chrome, Firefox или Safari. Проблемы могут возникать только у корпоративных пользователей.

    Если вы всё же вынуждены использовать Internet Explorer, следуйте этим рекомендациям по безопасности:

    • 🔒 Не используйте IE для посещения других сайтов — только для работы с ВТБ.
    • 🛡️ Установите дополнительный антивирус (например, Kaspersky Internet Security) и регулярно сканируйте систему.
    • 🔄 После работы с банком очищайте кэш и cookies: 
      ⚙ → Безопасность → Удалить журнал обозревателя → Выбрать "Файлы cookie" и "Временные файлы"

    FAQ: Частые вопросы по настройке Internet Explorer для ВТБ

    Можно ли использовать Internet Explorer на Windows 11 для ВТБ-Онлайн?

    На Windows 11 Internet Explorer удалён по умолчанию, но его можно запустить в режиме совместимости через Microsoft Edge:

    1. Откройте Edge и введите в адресной строке: edge://flags/#ie-mode.
    2. Включите параметр Internet Explorer mode.
    3. Добавьте online.vtb.ru в список сайтов для IE Mode через групповую политику или вручную.

    Если этот способ не работает, обратитесь в поддержку ВТБ для получения альтернативного решения (например, установки виртуальной машины с Windows 10).

    Почему после входа в ВТБ-Онлайн через IE появляется ошибка "Сессия истекла"?

    Эта ошибка связана с настройками куки (cookies) или кэша. Чтобы исправить:

    1. Откройте Свойства обозревателя → Конфиденциальность → Дополнительно.
    2. Установите галочку Автоматически обрабатывать файлы cookie и выберите Принимать для первых и третьих сторон.
    3. Очистите кэш (см. инструкцию выше).

    Если проблема сохраняется, попробуйте войти в личный кабинет в приватном режиме (Ctrl + Shift + P).

    Как настроить Internet Explorer для работы с ЭЦП в ВТБ для юридических лиц?

    Для юридических лиц требуется дополнительная настройка:

    1. Установите КриптоПро CSP и плагин CryptoPro CAdES Browser Plug-in (см. раздел выше).
    2. В IE разрешите запуск ActiveX: 
      Свойства обозревателя → Безопасность → Надёжные сайты → Уровень по умолчанию → Включить "Запуск элементов ActiveX"
    3. Добавьте в надёжные сайты адреса: 
      https://bank.vtb.ru

      https://cabinet.vtb.ru

    4. Если используете Java-апплеты, установите Java 8 Update 331 и разрешите её в панели управления Java.

    Для устранения ошибок проверьте инструкции ВТБ по ЭЦП.

    Что делать, если ВТБ-Онлайн просит обновить браузер, но я вынужден использовать IE?

    Это сообщение появляется, если сайт определяет устаревшую версию браузера. Обойти его можно двумя способами:

    1. Изменить User-Agent:

      Установите расширение User-Agent Switcher для IE (например, отсюда) и выберите IE 11 (Windows 10).

    2. Использовать IE Mode в Edge:

      Как правило, в этом режиме сайт ВТБ не показывает предупреждения об обновлении.

    Если ни один способ не помог, свяжитесь с технической поддержкой ВТБ по телефону 8 800 100-24-24 и уточните, какая версия браузера требуется для вашего типа доступа.

    Можно ли настроить Internet Explorer на Mac для работы с ВТБ?

    Internet Explorer никогда не выпускался для macOS. Альтернативные варианты:

    • 🍎 Используйте Safari или Chrome — для большинства клиентов ВТБ они поддерживаются официально.
    • 🖥️ Установите виртуальную машину с Windows 10 (например, через Parallels Desktop или VirtualBox).
    • 🌐 Попробуйте Microsoft Edge для Mac в режиме IE Mode (требуется настройка через групповую политику).

    Для корпоративных клиентов ВТБ может предоставить доступ к удалённому рабочему столу с предустановленным IE.
