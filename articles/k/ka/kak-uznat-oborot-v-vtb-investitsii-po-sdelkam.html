Как посмотреть оборот по сделкам в ВТБ Инвестиции: все способы с инструкциями

Оборот по сделкам в ВТБ Инвестиции — ключевой показатель для инвесторов, который помогает анализировать активность на бирже, рассчитывать налоговые обязательства или просто контролировать свои операции. Однако не все клиенты банка знают, где именно искать эту информацию: в мобильном приложении данные разбросаны по разным разделам, а в личном кабинете на сайте отчёты формируются по особым правилам. В этой статье разберём все доступные способы — от стандартных выписок до скрытых функций API, которые используют опытные трейдеры.

Особенность ВТБ Инвестиции в том, что оборот здесь учитывается не только по покупке/продаже ценных бумаг, но и по другим операциям: дивидендам, купонным выплатам, комиссиям. При этом банк не всегда показывает суммарный оборот «одной кнопкой» — его нужно собирать из нескольких источников. Мы собрали 5 проверенных методов, включая малоизвестные лайфхаки, которые экономят время.

1. Оборот в мобильном приложении ВТБ Инвестиции: где искать

Самый быстрый способ — использовать официальное приложение ВТБ Инвестиции (доступно для iOS и Android). Однако здесь есть нюанс: приложение не показывает оборот напрямую, но его можно рассчитать через раздел «История операций». Вот пошаговая инструкция:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите в Портфель → История операций.
  3. Нажмите на фильтр (значок воронки) и выберите период (например, «За год»).
  4. Включите галочки напротив «Сделки», «Дивиденды» и «Комиссии».
  5. Экспортируйте данные в Excel (кнопка «Экспорт» в правом верхнем углу).

В полученной таблице оборот по сделкам будет равен сумме столбцов Сумма сделки (руб.) по операциям «Покупка» и «Продажа». Важно: в приложении не учитываются сделки по внебиржевым инструментам (например, структурным продуктам) — их нужно проверять отдельно через личный кабинет.

2. Личный кабинет ВТБ Онлайн: скрытые отчёты

Веб-версия ВТБ Онлайн предлагает более гибкие инструменты для анализа оборота, чем мобильное приложение. Чтобы найти данные:

  1. Зайдите на сайт online.vtb.ru и авторизуйтесь.
  2. Перейдите в Инвестиции → Отчёты → Отчёт по сделкам.
  3. Выберите нужный брокерский счёт и период (максимум — 3 года).
  4. Нажмите Сформировать отчёт и дождитесь генерации PDF-файла.

В отчёте оборот будет указан в двух разделах:

  • 📊 «Сводка по сделкам» — сумма всех покупок и продаж за период.
  • 💰 «Денежные потоки» — оборот с учётом комиссий и налогов.

⚠️ Внимание: В отчётах ВТБ Онлайн не отображаются сделки по ИИС (индивидуальным инвестиционным счётам) старше 3 лет. Для таких случаев придётся запрашивать выписку через поддержку.

3. Выписка по брокерскому счёту: детальный разбор

Если нужны данные для налоговой декларации или аудита, лучший вариант — официальная выписка по счёту. Её можно получить двумя способами:

  • 📄 Через личный кабинет: Инвестиции → Документы → Выписки по счёту. Выберите период и формат (PDF или Excel).
  • 🏦 В отделении банка: Закажите выписку у менеджера (может потребоваться паспорт).

В выписке оборот отражён в столбцах:

  • 🔹 Дата операции — для фильтрации по периоду.
  • 🔹 Тип операции («Покупка», «Продажа», «Дивиденды»).
  • 🔹 Сумма (руб.) — оборот по каждой сделке.

Тип операции Учитывается в обороте? Примечание
Покупка/продажа акций Да Сумма сделки без учёта комиссии
Дивиденды Нет Учитываются в доходе, но не в обороте
Комиссия брокера Да Включается в общий оборот
Купонные выплаты по облигациям Нет Доход, а не оборот

⚠️ Внимание: В выписке по ИИС оборот по сделкам может дублироваться в разделе «Налоговые данные». Не суммируйте эти строки дважды!

4. API ВТБ Инвестиции: автоматический сбор данных

Для опытных инвесторов и разработчиков есть способ получать оборот программно — через API ВТБ Инвестиции. Банк предоставляет доступ к данным по сделкам через OpenAPI, но для этого нужно:

  1. Получить API-ключ в личном кабинете (Настройки → Безопасность → API).
  2. Использовать endpoint /operations с параметрами: 
    {

    "from": "2026-01-01",
    

    "to": "2026-12-31",
    

    "brokerAccountId": "ваш_номер_счёта"
    

    }

  3. Обработать ответ, где оборот будет в полях totalAmountBuy и totalAmountSell.

Пример ответа API (упрощённо):

{

"operations": [


{


"date": "2026-05-15",


"type": "Buy",


"amount": 50000.00,


"instrument": "SBER"


},


{


"date": "2026-05-20",


"type": "Sell",


"amount": 52000.00,


"instrument": "SBER"


}


],


"totalTurnover": 102000.00


}

🔧 Технический нюанс: API ВТБ имеет лимит запросов — 1000 операций за вызов. Для больших портфелей придётся использовать пагинацию (limit и offset).

Как получить расширенный доступ к API?

Для работы с историческими данными старше 1 года нужно написать запрос в поддержку ВТБ с указанием цели (например, "автоматизация налоговой отчётности"). Банк может предоставить доступ к архивным endpointам, но это занимает до 5 рабочих дней.

5. Альтернативные сервисы: Tinkoff Invest, Мой Брокер

Если вам не хватает функционала ВТБ, можно подключить брокерский счёт к сторонним сервисам анализа. Они автоматически рассчитывают оборот и визуализируют данные:

  • 📈 Tinkoff Invest (через импорт выписки): показывает оборот в разделе Аналитика → Статистика.
  • 📊 Мой Брокер (moybroker.ru): формирует отчёты с детализацией по инструментам.
  • 🔍 InvestHero: агрегирует данные из нескольких брокеров, включая ВТБ.

Для импорта данных в эти сервисы:

  1. Экспортируйте выписку из ВТБ в формате .xlsx или .csv.
  2. Загрузите файл в выбранный сервис (обычно раздел Импорт данных).
  3. Дождитесь обработки (может занять до 10 минут).

⚠️ Внимание: Сторонние сервисы могут не учитывать внебиржевые сделки ВТБ (например, по структурным нотам). Всегда сверяйте их данные с официальными выписками!

💡

Если используете Google Sheets, можно автоматизировать сбор оборота через формулу =IMPORTDATA("URL_выписки") (предварительно конвертируйте выписку в CSV и загрузите в Google Drive).

6. Частые ошибки при расчёте оборота

Даже опытные инвесторы допускают ошибки при подсчёте оборота. Вот самые распространённые:

  • 🔄 Двойной счёт комиссий: Некоторые суммируют комиссии отдельно и включают их в оборот повторно.
  • 📅 Неправильный период: Например, берут данные за календарный год вместо налогового (с января по декабрь vs. с апреля по март).
  • 💱 Игнорирование валютных операций: Оборот по сделкам в долларах нужно пересчитывать по курсу ЦБ на дату операции.
  • 📄 Пропуск внебиржевых сделок: Структурные продукты или ОФЗ могут не отображаться в стандартных отчётах.

Чтобы избежать ошибок:

  1. Всегда сверяйте данные из двух источников (например, приложение + выписка).
  2. Используйте Excel для автоматического суммирования: =СУММЕСЛИ(диапазон; "Покупка"; диапазон_сумм).
  3. Для ИИС проверяйте оборот в разделе Налоговые данные — там могут быть скрытые корректировки.

💡

Оборот по сделкам ≠ налоговая база. Для декларации 3-НДФЛ нужно учитывать только прибыльные операции (продажи с доходом).

FAQ: Ответы на популярные вопросы

🔍 Можно ли увидеть оборот по конкретной акции (например, только по Сбербанку)?

Да, в личном кабинете ВТБ Онлайн перейдите в Инвестиции → Портфель → История сделок, выберите фильтр по инструменту (SBER) и экспортируйте данные. В Excel отсортируйте по типу операции («Покупка»/«Продажа») и просуммируйте столбец Сумма.

⏳ Сколько хранится история сделок в ВТБ Инвестиции?

В мобильном приложении — до 3 лет, в личном кабинете ВТБ Онлайн — до 5 лет. Для более старых данных нужно запрашивать архивную выписку через поддержку (обработка занимает до 10 дней).

💸 Почему оборот в выписке не совпадает с данными из приложения?

Причины расхождений:

  • В приложении не учитываются внебиржевые сделки (ОФЗ, структурные продукты).
  • Выписка может включать корректировки (например, по аннулированным сделкам).
  • В некоторых версиях приложения не отображаются сделки по ИИС старше 1 года.

Всегда ориентируйтесь на данные из официальной выписки.

📱 Можно ли получить оборот через чат-бот ВТБ?

Нет, чат-бот в Telegram или ВКонтакте (@VTB_Invest_Bot) показывает только баланс счёта и текущий портфель, но не оборот по сделкам. Для этой цели подходят только личный кабинет, приложение или API.

🔐 Нужно ли подтверждать личность для получения выписки с оборотом?

Если запрашиваете выписку через личный кабинет или приложение — нет. При обращении в отделение банка потребуется паспорт. Для API-доступа нужна двухфакторная аутентификация (СМС или push-уведомление).
